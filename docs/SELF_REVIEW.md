# v0.6.0 自检记录

## 本轮目标

- 首次打开必须有清晰介绍，不再让用户不知道如何开始生成真实网页海报。
- 真实截图并发数量可调节为 1–63。
- 设置页突出“海报截图设置”，并说明高并发的代价。
- 全界面图标统一，避免继续使用杂乱的 Unicode 符号。
- 保持海报墙为主界面核心，AI 默认关闭。

## 已完成

1. 首次引导升级为欢迎向导：包含产品说明、三步流程、截图引擎选择、并发数量滑杆、隐私说明和立即生成入口。
2. 并发设置从固定 1–4 扩展为 1–63，内部通过 `MAX_SCREENSHOT_CONCURRENCY = 63` 和 `clampConcurrency()` 做保护。
3. 设置页“海报截图设置”加入 1–63 滑杆与实时数字显示。
4. 新增本地 SVG 图标系统 `iconSvg()` / `installStaticIcons()`，覆盖侧边栏、顶部栏、设置页、批量操作栏、文件夹树和状态标签。
5. 重新绘制扩展图标 PNG，避免依赖外部 iconfont 资源和版权不确定性。
6. 更新测试用例，覆盖版本、并发范围、首次引导、图标系统、海报化管线和 Manifest 权限。
