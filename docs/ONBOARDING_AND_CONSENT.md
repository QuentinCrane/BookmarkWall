# 首次引导与截图授权说明（v0.7.7）

从 v0.7.7 开始，插件首次打开时只展示书签和本地兜底封面，不会自动并发访问网页生成真实截图。

用户需要在首次引导中完成以下动作后，插件才会开始真实海报生成：

1. 阅读“真实海报生成会访问网页并并发加载页面”的说明。
2. 勾选“我同意开始后自动并发生成当前可见海报”。
3. 点击“开始生成真实海报”。

技术实现：

- 使用 `settings.onboarding.posterConsentGranted` 记录用户是否同意。
- `init()` 中的初始自动生成入口受 `hasPosterGenerationConsent()` 保护。
- `scheduleVisiblePosterGeneration()` 受同一授权开关保护。
- 手动点击“生成当前可见海报”时，如未授权，会重新打开引导并提示用户确认。

这样可以避免在用户未理解截图行为和并发数量前，插件自动打开网页并生成截图。
