# 隐私与权限说明

## 本地优先

书签海报墙直接读取浏览器原生书签，并在本地完成展示和整理。插件不会将书签、截图、缓存、AI API Key 上传到开发者服务器。

## 真实截图数据

真实网页截图用于生成书签海报，保存到本地扩展存储中。用户可在设置页清除全部海报缓存。

## 权限用途

- `bookmarks`：读取、移动、重命名和删除浏览器原生书签。
- `storage` / `unlimitedStorage`：保存设置、海报缓存、AI 建议。
- `tabs`：打开截图标签页、手动兼容补拍窗口、恢复原标签页、打开或切回扩展主页面。
- `tabGroups`：把批量截图临时标签页收进折叠的“BookmarkWall 截图工作区”，减少标签栏干扰。
- `activeTab`：手动当前页截图备用。
- `scripting`：截图前滚动网页到顶部，避免截到中间位置。
- `debugger`：使用 Chrome DevTools Protocol 生成真实网页截图。
- `http://*/*` / `https://*/*`：加载书签网页并生成截图或读取公开封面。

## AI 预整理

AI 默认关闭。开启后，只有用户主动点击 AI 预整理才会发送选中书签信息到用户配置的 AI 服务商。默认发送字段包括：标题、域名、URL 路径、当前文件夹、已有文件夹列表。不会发送浏览历史、访问次数、Cookie、页面截图或完整书签库。


## 书签备份隐私说明

全部书签导出仅在本机浏览器中生成下载文件，不会上传到开发者服务器。HTML 备份用于导回 Chrome / Edge，JSON 备份用于用户自行保存和排查。
