# 图标系统说明

v0.6.0 起插件不依赖外部 iconfont 或远程图标库。界面图标使用本地内置 SVG path，由 `src/app.js` 中的 `iconSvg()` 统一渲染。

这样做的原因：

1. 避免外部图标版权和授权不确定性；
2. 避免扩展运行时加载远程资源，符合 MV3 的安全边界；
3. 图标颜色、尺寸、描边可以统一受 CSS 控制；
4. 打包后离线可用。

当前覆盖范围：

- 应用 Logo；
- 侧边栏智能入口；
- 文件夹树；
- 顶部搜索和操作按钮；
- 批量操作栏；
- 设置页导航；
- 首次引导；
- 卡片底部状态标签。

后续如需替换为 iconfont / Alibaba Iconfont，需要确认授权方式，并将 SVG 下载到本地 assets 后再打包，不应在扩展运行时远程加载。
